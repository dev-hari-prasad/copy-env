# Contributing

This repository follows a **strict contribution** policy to protect security and predictably for users.

In most cases, code contributions from **external contributors are not accepted**.
Considering the likely security implications of this project, **contributions are limited to non-code contributions such as bug report, security report, and improvement suggestions through issues.**


If you find an issue, bug, or vulnerability:
1. Open an issue on GitHub: https://github.com/dev-hari-prasad/copy-env/issues
2. Or contact the maintainer on: webdev.byhari@gmail.com

**If you have a suggestion for improvement**, please open an issue with your suggestion and any relevant details.
